# Backend Team360

Backend Python base para Team360, alineado con el stack operativo de Litestar y `uv` usado como referencia en Vertice360.

Incluye:
- rutas iniciales de salud, AG-UI y Team360
- nucleo `team360_orquestador`
- soporte estructural para `agui_stream`
- providers iniciales de mensajeria: `gupshup` y `mercadolibre`
- laboratorio de browser con `playwright` para Mercado Libre
