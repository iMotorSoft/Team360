# SAP Business One Y ERP

SAP Business One es un vertical premium para Team360. La oportunidad no es reemplazar SAP B1, sino agregar una capa de inteligencia operativa, consulta, alertas, reportes y automatizacion asistida.

## Fase Inicial Recomendada

Primero lectura, descarga de reportes, consultas de stock, clientes, deuda, pedidos y pre-carga con aprobacion humana.

No iniciar con facturacion automatica, pagos, asientos contables, cambios masivos de precios o escritura irreversible.

## RPA Desktop Asistido

RPA Desktop sobre sesion autorizada del usuario puede validar valor rapido, con logs, screenshots y supervision humana.

Debe tratarse como automatizacion asistida, no autonomia 24/7.

## Service Layer Y DI API

Service Layer y DI API son caminos mas robustos cuando el cliente acepta configuracion tecnica, permisos y revision de licenciamiento.

Para escritura estructurada, preferir helper controlado y aprobaciones.

## Reglas

ERP con escritura critica requiere approval_required o consultoria previa.

ERP read-only puede entrar en automatizacion operativa si hay permisos claros y volumen suficiente.
