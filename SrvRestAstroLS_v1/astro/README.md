# Team360 Astro Frontend

Base tecnica inicial del frontend Team360.

## Stack

- Astro 6
- Svelte 5 con Runes
- Tailwind CSS 4 mediante `@tailwindcss/vite`
- DaisyUI 5 CSS-first con tema neutral `team360`
- TypeScript strict
- pnpm

## Comandos

```bash
pnpm install
pnpm dev
pnpm build
pnpm check
```

## Alcance de Fase 1

Incluye scaffold compilable, estilos globales, tema `team360`, wrappers UI minimos y reserva estructural para AG-UI.

No incluye App Shell, autenticacion, navegacion contextual, rutas de consola ni integracion backend funcional.
