# Documentacion Frontend Team360

Documentacion tecnica de frontend, arquitectura UX, estrategias de migracion y definiciones de stack.

## Documentos

- `team360-frontend-technical-base-from-vertice360.md`: analisis de Vertice360 como base tecnica/UX para Team360, deteccion de brechas, propuesta de stack moderno (Astro latest, Svelte 5, Tailwind 4, DaisyUI 5, AG-UI) y estrategia de migracion por fases.
- `team360-package-manager-and-ui-policy.md`: politica obligatoria de pnpm, Tailwind CSS 4 + DaisyUI 5 CSS-first y wrappers UI propios.
